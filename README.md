# Login Page UI

A simple and responsive Login Page UI built using HTML and CSS.  
This project demonstrates modern UI design principles, responsive layouts, clean form structure, and interactive styling effects.

---

## 📌 Project Objective

The objective of this project is to create a visually appealing and fully responsive login interface using only HTML and CSS.

The project focuses on:

- Clean user interface design
- Proper alignment and spacing
- Responsive layout for different screen sizes
- Modern styling with gradients and shadows
- User-friendly form structure

---

## 🚀 Features

- Responsive Login Form
- Modern Gradient Background
- Input Focus Effects
- Hover Animation on Button
- Mobile-Friendly Design
- Clean and Minimal UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
